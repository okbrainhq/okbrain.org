This is a reference implementation of an AI Harness based on the OKBrain philosophy. It is released under the MIT License — you can use it for any purpose without attribution. The source code is available on [GitHub](https://github.com/okbrainhq/OKBrain-Harness).

We do not accept contributions in the form of pull requests or issue reports. However, if you find a security issue, you can notify us at [hello@okbrain.org](mailto:hello@okbrain.org).

## Why?

We believe coding agents are extremely good at building new features and fixing bugs. You can simply describe what you need and they will do it. For the best results, use Claude Code Opus 4.6 or GPT 5.4. Even open-source models like GLM5 and Kimi K2.5 do a very good job.

Since we have an extensive e2e test suite, coding models can validate their fixes and get the feedback they need to iterate and implement what you have asked for.

## Future

We believe coding is already a solved problem with coding agents. In the coming months, they will get so good and fast that there will be very little demand for human collaboration when building software. A team of AI models will work together to build the right tools for most of us.

We are just getting ready for it.
