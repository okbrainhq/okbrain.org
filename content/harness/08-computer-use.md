---
title: Computer Use
---

AI models can use the computer in a non-root environment to do anything they need. We also include common data science and utility libraries, so they can handle tasks like data analysis, audio and video processing, and PDF analysis.

![OKBrain computer use terminal interface](/images/okbrain-computer-use-1.webp)
![OKBrain executing computer commands](/images/okbrain-computer-use-2.webp)

Models can also install tools into the user space. Currently, they can only work with CLI tools.

We believe AI agents can accomplish most tasks using CLIs and headless tools. That said, adding GUI support is not difficult — you can ask a coding agent to add it.
