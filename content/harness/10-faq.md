# Frequently Asked Questions

## What is Harness?

Harness is the open-source orchestration layer for running AI agents at scale.

## Is it free?

Yes. Harness is open-source under the MIT license.

## Which LLM providers are supported?

OpenAI, Anthropic, and Ollama (local) are supported out of the box.

## Can I use my own tools?

Yes. Any async JavaScript function can be used as a step in a pipeline.
